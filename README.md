# Unconventional magnetism mediated by spin-phonon-photon coupling

This project produces the figures appearing on the above-mentioned article. More specifically one can find

- Multi-layer Green functions
- Magnetoelastic coupling
- Phase transition analysis
- Figure reproduction (Fig2, Fig3)

## Dependencies

- CMake
- Eigen
- OpenCV
- Python (for matplotlib-cpp, optional)

## Run
./unconventional_figures

## Build
```bash
mkdir build
cd build
cmake ..
cmake --build
